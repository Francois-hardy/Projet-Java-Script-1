Informations

Envie de folles vitesses ? Prenez le volant d'une Formule 1 et négociez les chicanes sans sortir de la piste. Attention aux virages !

Comment jouer ?

Déplacez votre F1 avec les flêches directionnels de votre clavier. Evitez de sortir de la piste, ce qui endommage votre voiture.

Les touches

Droite : -> Gauche : <- Pause : P
